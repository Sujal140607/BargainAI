# 🤖 BargainAI

> **Negotiate Smarter, Pay Less.**

BargainAI is an AI-powered negotiation game where players bargain with an intelligent AI seller to purchase products at the lowest possible price.

The AI has hidden constraints such as minimum selling price, target profit, patience, trust level, and negotiation strategy. Your goal is to convince the seller to offer the best possible deal.

---

## 🚀 Features

- 🔐 JWT Authentication
- 💬 AI Negotiation Chat
- 🤖 Intelligent AI Seller
- ⚡ Real-time Communication (Socket.io)
- 🧠 LangChain Integration
- 🏆 Global Leaderboard
- 📈 Negotiation Analytics
